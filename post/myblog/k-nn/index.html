K近邻法

Nov 11, 2023

1. $k$ 近邻法是什么？

$k$ 近邻法（$k$-nearest neighbor, $k$-NN）是一种基本且简单的分类和回归方法。
$k$ 近邻法简单直观来说：给定一个训练数据集，对于新的输入实例，在训练数据集重找到与该实例最接近的 $k$ 个实例，这 $k$ 个实例的多数属于哪个类，就把该输入实例分入这个类。
$k$ 近邻法有三个基本要素：$k$ 值的选择，距离度量和分类决策规则。

2. 三个基本要素

2.1. $k$ 值的选择

$k$ 的选择会有重大影响，较小则整体模型变复杂，对噪声会非常敏感，容易过拟合；较大则模型变简单，预测误差会增大。

2.2. 距离度量

要找到 $k$ 个实例，就要指定一个范围。定义： $$ L_p(x_i,x_j)=(\sum_{l=1}^{n}|x_i^{(l)}-x_j^{(l)}|^p)^{\frac{1}{p}} $$

  • 其中 $p \geq 1$
  • 当 $p=2$ 称为欧氏距离（Euclidean distance）
  • 当 $p=1$ 称为曼哈顿距离（Manhattan distance）
  • 当 $p=\infty$ 是各个坐标距离的最大值

2.3. 分类决策规则

一般是选择多数表决，即为统计出现的不同类别的次数，选择出现次数最多的类别作为新数据点的分类。

3. $k$ 近邻法的实现原理

$k$ 近邻法的实现需要考虑如何快速搜索 $k$ 个最近邻点。$kd$ 树是一种便于对 $k$ 维空间的数据进行快速检索的数据结构。 $kd$ 树是一种二叉树，表示对 $k$ 维空间的一个划分，其每个节点对应于 $k$ 维空间划分重的一个超矩形区域。利用 $kd$ 树可以省区对大部分数据的搜索，减少计算量

下面是kd树的基本思想和构建过程：

  1. 构建kd树：kd树的构建是一个递归过程，它选择一个轴（通常是特征空间的一个维度），然后选择该轴上的一个中位数作为分割点，将数据集分成两个子集。左子树包含小于中位数的数据，右子树包含大于中位数的数据。然后递归地构建左子树和右子树，每个子树选择下一个轴和中位数。这个过程一直进行，直到每个子树中包含一个数据点或没有数据点。

  2. kd树的平衡性：构建kd树时，通常要确保树保持平衡，以获得最佳查询性能。平衡性意味着每个节点都有大致相等数量的左右子树节点

  3. 查询kd树：一旦kd树构建完成，可以使用它来执行k-NN查询或范围查询。对于k-NN查询，从根节点开始，沿着树的分支向下遍历，根据查询点与分割超平面的距离来选择向左子树或右子树移动。这个过程一直进行，直到找到k个最近邻或遍历完整个树。对于范围查询，可以确定是否需要搜索每个子树，从而过滤掉一些不必要的子树。

4. 代码实现

def distance(a, b):
	return np.sqrt(np.sum(np.square(a - b)))
  • KNN 算法
class KNN:
	# 初始化
	def __init__(self, k, label_num):
		self.k = k
		self.label_num = label_num # 类别的数量
	
	# 保存训练数据
	def fit(self, x_train, y_train):
		self.x_train = x_train
		self.y_train = y_train
	
	# 获取距离目标样本点最近的K个样本点的标签 
	def get_knn_indices(self, x): 
		# 计算已知样本的距离 
		dis = list(map(lambda a: distance(a, x), self.x_train)) 
		# 按距离从小到大排序，并得到对应的下标 
		knn_indices = np.argsort(dis) 
		# 取最近的K个 
		knn_indices = knn_indices[:self.k] 
		return knn_indices

	# 观测K个近邻并使用np.argmax获取其中数量最多的类别
	def get_lable(self, x):
		# 获取选择的K个样本点
		knn_indices self.get_knn_indices(x)
		# 类别计数
		label_statistic = np.zeros(shape=[self.;label_num])
		for index in knn_indices:
			label = int(self.y_train[index])
			label_statistic[label] += 1
		# 返回数量最多的类别
		return np.argmax(lable_statistic)
	
	# 预测样本 test_x 的类别
	def predict(self, x_test):
			predicted_test_labels = np.zeros(shape=[len(x_test)], dtype=int)
		for i, x in enumerate(x_test):
			predict_test_labels[i] = self.get_label(x)
		return predicted_test_labels
